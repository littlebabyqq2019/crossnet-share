## 💡 核心设计

### 架构模式
```
方案1增强版：A作为文件代理服务器 + 轻量级客户端

优点：
✓ 实现简单，逻辑清晰
✓ 不需要修改网络配置
✓ 集中管理和权限控制
✓ 安全性可控

适用场景：
✓ 小文件（<10MB），数量多（数百到数千）
✓ 按日期批量下载
✓ 局域网千兆网络
```

### 技术栈
- **语言**: C++17
- **GUI框架**: Qt5 (Core, Widgets, Network)
- **JSON库**: nlohmann/json (header-only)
- **构建工具**: CMake 3.15+
- **网络通信**: TCP Socket (Qt Network)

### 通信协议
- **格式**: [4字节长度] + [JSON消息体]
- **消息类型**: 12种（注册、文件列表、下载、上传、批量传输等）
- **文件传输**: 分块传输（64KB/块）
- **编码**: UTF-8 JSON + Base64（二进制数据）

## 🚀 使用流程

### 1. 编译项目
```powershell
mkdir build && cd build
cmake .. -DCMAKE_PREFIX_PATH="C:\Qt\5.15.2\msvc2019_64"
cmake --build . --config Release
```

### 2. 部署到3台电脑

**A电脑（服务器）**:
- 运行 `CrossNetShareServer.exe`
- 配置端口 8888，启动服务

**B电脑（客户端1）**:
- 运行 `CrossNetShareClient.exe`
- 连接到 `192.168.1.100:8888`
- 注册为 "ClientB"，共享 `D:\SharedFiles`

**C电脑（客户端2）**:
- 运行 `CrossNetShareClient.exe`
- 连接到 `10.28.168.100:8888`
- 注册为 "ClientC"，共享 `E:\MyFiles`

### 3. 跨网段文件共享

**B电脑下载C的文件**:
1. 选择日期范围
2. 点击"Refresh"查看C的文件列表
3. 点击"Download Selected"批量下载

**C电脑上传文件给B**:
1. 点击"Upload File"
2. 选择本地文件上传
3. B电脑刷新即可看到新文件
